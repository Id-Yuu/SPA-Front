##### How to Use
```
git clone https://github.com/Id-Yuu/SPA-Front.git --branch 01-Starter-SPA --single-branch
cd SPA-Front
npm install
npm run dev
```

- React and React-Router
- Consume API Github
- Sass
- ViteJs
